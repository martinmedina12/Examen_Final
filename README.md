Pregunta 3
El scope es un orden que se sigue el ejecutar el programa, primero se buscas variblas locales y despues globales. Además, delimita en donde puede existir una variable. Una variable local solo se encuentra dentro de una función y para usarla, se llama la función, mientras la variable global se encuentra en todo el programa pudiendo acceder a ella desde cualquier lugar.
